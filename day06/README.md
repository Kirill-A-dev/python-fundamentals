# Day 06 — Dictionaries and Filtering Students

## What I practiced

- Working with dictionaries
- Accessing dictionary values by key
- Using `.keys()`, `.values()`, `.items()`
- Looping through a list of dictionaries
- Filtering data with `if / elif / else`
- Appending values to new lists
- Counting list items with `len()`
- Printing formatted output

## Project

`student_filters.py`

The program stores students as dictionaries inside a list.

Each student has:

- name
- grade

The program separates students into three groups:

- Excellent students
- Good students
- Students who need improvement

It also prints a summary with the number of students in each group and the total number of students.
