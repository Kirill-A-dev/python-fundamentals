# Day 7 — Nested Data Structures

## Topics

- List of dictionaries
- Dictionary values as lists
- Accessing nested data
- Calculating average values with `sum()` and `len()`
- Rounding numbers with `round()`
- Assigning status with `if / elif / else`
- Formatted output with f-strings

## Project

Student average report.

The program stores students as dictionaries.  
Each student has a name and a list of grades.  
The program calculates the average grade and assigns a status.
