# Day 09 — Log Analyzer Practice

## Goal

Practice basic log analysis using Python data structures.

The script analyzes a list of log messages, extracts the log level from each line, counts how many times each level appears, and prints a summary report.

## Topics practiced

- Lists
- Strings
- `split()`
- Dictionaries
- `for` loops
- `if level in stats`
- `dict.items()`
- Basic log analysis logic

## Files

- `access_log_analyzer.py` — analyzes logs stored directly inside the Python script.
