## Project description

This project is a simple command-line grade manager.

The program allows the user to:

- enter student names;
- enter grades from 1 to 5;
- validate incorrect input;
- stop the program with `stop`;
- print a grade report;
- calculate the average grade;
- show students with the highest grade;
- show students with the lowest grade.

## Example output
```text
--- Grade Report ---
Alex - 2
Kirill - 2
Anna - 3
Max - 4
Inna - 5
Leo - 5

Total students: 6

Average grade: 3.5
Best students: Inna, Leo - 5
Worst students: Alex, Kirill - 2
```
